# TimeForge: Echoes of Infinity

A cosmic idle/incremental game where you forge entire universes, collapse them into Echo Crystals, and build ever more powerful cosmic creations.

## Game Features

### Core Gameplay
- **Forge Clicking**: Click the cosmic forge to generate Sparks, your basic currency
- **Building System**: Purchase buildings that automatically generate Flux and Civilization
- **Idle Progression**: Buildings continue producing even when you're away
- **Universe Collapse**: End your current universe to earn permanent Echo Crystals

### Roguelite Elements
- **Universe Traits**: Each run starts with unique traits that modify gameplay
- **Trait Rarities**: Common, Uncommon, Rare, and Mythic traits with different power levels
- **Strategic Choices**: Select traits at the start of each universe to shape your strategy

### Progression Systems
- **Permanent Upgrades**: Spend Echo Crystals on upgrades that persist across all universes
- **Run Upgrades**: Temporary upgrades that enhance your current universe
- **Achievements**: Unlock achievements to earn Shards and track milestones
- **Cross-Device Sync**: Sign in to save progress and play across multiple devices

## Technical Stack

- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth)
- **Build Tool**: Vite
- **Icons**: Lucide React

## Getting Started

1. The game starts with Universe #1 and a trait selection screen
2. Choose up to 2 traits to modify your universe's rules
3. Click the Forge button to generate Sparks
4. Buy buildings to automate production
5. Purchase upgrades to boost your efficiency
6. When ready, collapse your universe to earn Echo Crystals
7. Use Echo Crystals for permanent upgrades
8. Start a new universe and repeat with different traits!

## Game Mechanics

### Resources
- **Sparks**: Generated by clicking, used to buy buildings
- **Flux**: Generated by buildings, main production resource
- **Civilization**: Advanced resource for late-game progression
- **Echo Crystals**: Meta-currency earned by collapsing universes
- **Shards**: Earned from achievements (future: used for cosmetics)

### Buildings
- **Foundry**: Basic Flux generator
- **Reactor**: Enhanced Flux production
- **Laboratory**: Advanced Flux generation
- **Civilization Seed**: Generates Civilization
- **World Engine**: Massive Flux producer
- **Megastructure**: Advanced Civilization generator

### Universe Traits Examples
- **Arcane Physics**: Labs +200%, Reactors -30%
- **Quantum Flux**: All Flux +300%
- **Entropy Plus**: All production +100%, runs end sooner
- **Philosopher's Stone**: Click power +500%
- And many more to discover!
